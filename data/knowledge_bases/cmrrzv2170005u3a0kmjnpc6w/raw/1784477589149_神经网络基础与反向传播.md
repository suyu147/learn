# 神经网络基础与反向传播

## 📋 学习目标
- 理解神经网络的基本结构
- 掌握常用激活函数及其特性
- 理解反向传播算法的原理
- 能够手写实现简单的神经网络

## 📖 核心概念

### 神经网络结构

一个典型的神经网络由以下层组成：
1. **输入层**：接收原始数据
2. **隐藏层**：进行特征提取和变换
3. **输出层**：产生最终预测结果

### 前向传播

前向传播是从输入层到输出层的计算过程：
$$ z_i = \mathbf{W}_i \mathbf{a}_{i-1} + \mathbf{b}_i $$
$$ a_i = \sigma(z_i) $$

其中：
- $z_i$：第 i 层的加权输入
- $\mathbf{W}_i$：第 i 层的权重矩阵
- $\mathbf{b}_i$：第 i 层的偏置向量
- $\sigma$：激活函数
- $a_i$：第 i 层的激活输出

### 激活函数

| 激活函数 | 表达式 | 特点 |
|---------|--------|------|
| Sigmoid | $\sigma(x) = \frac{1}{1 + e^{-x}}$ | 将输出压缩到 (0, 1)，用于二分类 |
| Tanh | $\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$ | 将输出压缩到 (-1, 1)，零均值 |
| ReLU | $\text{ReLU}(x) = \max(0, x)$ | 计算简单，缓解梯度消失 |
| Leaky ReLU | $\text{LeakyReLU}(x) = \max(\alpha x, x)$ | 解决 ReLU 神经元死亡问题 |

### 损失函数

常见的损失函数：
- **均方误差 (MSE)**：用于回归任务
  $$ \text{MSE} = \frac{1}{n} \sum_{i=1}^n (y_i - \hat{y}_i)^2 $$

- **交叉熵损失**：用于分类任务
  $$ \text{CrossEntropy} = -\frac{1}{n} \sum_{i=1}^n y_i \log(\hat{y}_i) $$

### 反向传播算法

反向传播利用链式法则计算梯度：

1. **输出层梯度**：
   $$ \delta_L = \frac{\partial L}{\partial \hat{y}} \odot \sigma'(z_L) $$

2. **隐藏层梯度**：
   $$ \delta_i = (\mathbf{W}_{i+1}^T \delta_{i+1}) \odot \sigma'(z_i) $$

3. **权重和偏置的梯度**：
   $$ \frac{\partial L}{\partial \mathbf{W}_i} = \delta_i \mathbf{a}_{i-1}^T $$
   $$ \frac{\partial L}{\partial \mathbf{b}_i} = \delta_i $$

### 梯度下降优化

$$ \mathbf{W}_i = \mathbf{W}_i - \eta \frac{\partial L}{\partial \mathbf{W}_i} $$
$$ \mathbf{b}_i = \mathbf{b}_i - \eta \frac{\partial L}{\partial \mathbf{b}_i} $$

其中 $\eta$ 是学习率。

## 💻 代码实现

```python
import numpy as np

class SimpleNN:
    def __init__(self, input_size, hidden_size, output_size):
        # 初始化权重和偏置
        self.W1 = np.random.randn(input_size, hidden_size) * 0.01
        self.b1 = np.zeros((1, hidden_size))
        self.W2 = np.random.randn(hidden_size, output_size) * 0.01
        self.b2 = np.zeros((1, output_size))
    
    def sigmoid(self, z):
        return 1 / (1 + np.exp(-z))
    
    def sigmoid_derivative(self, z):
        return z * (1 - z)
    
    def forward(self, X):
        # 前向传播
        self.z1 = np.dot(X, self.W1) + self.b1
        self.a1 = self.sigmoid(self.z1)
        self.z2 = np.dot(self.a1, self.W2) + self.b2
        self.a2 = self.sigmoid(self.z2)
        return self.a2
    
    def backward(self, X, y, output, learning_rate=0.01):
        # 反向传播
        m = X.shape[0]
        
        # 输出层梯度
        delta2 = (output - y) * self.sigmoid_derivative(output)
        dW2 = np.dot(self.a1.T, delta2) / m
        db2 = np.sum(delta2, axis=0, keepdims=True) / m
        
        # 隐藏层梯度
        delta1 = np.dot(delta2, self.W2.T) * self.sigmoid_derivative(self.a1)
        dW1 = np.dot(X.T, delta1) / m
        db1 = np.sum(delta1, axis=0, keepdims=True) / m
        
        # 更新权重和偏置
        self.W1 -= learning_rate * dW1
        self.b1 -= learning_rate * db1
        self.W2 -= learning_rate * dW2
        self.b2 -= learning_rate * db2
    
    def train(self, X, y, epochs=1000, learning_rate=0.01):
        for epoch in range(epochs):
            output = self.forward(X)
            loss = np.mean((output - y) ** 2)
            self.backward(X, y, output, learning_rate)
            if epoch % 100 == 0:
                print(f"Epoch {epoch}, Loss: {loss:.4f}")

# 使用示例
nn = SimpleNN(input_size=2, hidden_size=4, output_size=1)
X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
y = np.array([[0], [1], [1], [0]])
nn.train(X, y, epochs=5000, learning_rate=0.1)
print("预测结果:", nn.forward(X))
```

## 🎯 关键要点
- 反向传播是训练神经网络的核心算法
- 梯度消失问题：当网络很深时，梯度可能趋近于零
- 梯度爆炸问题：梯度可能指数级增长
- ReLU 激活函数有助于缓解梯度消失问题

## ❓ 常见问题

Q: 为什么需要激活函数？
A: 激活函数引入非线性，否则神经网络只是线性变换的叠加，无法学习复杂模式。

Q: 梯度消失如何解决？
A: 使用 ReLU 激活函数、残差连接、批归一化等方法。
