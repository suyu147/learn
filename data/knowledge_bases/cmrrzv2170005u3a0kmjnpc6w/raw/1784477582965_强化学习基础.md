# 强化学习基础

## 📋 学习目标
- 理解强化学习与监督学习的区别
- 掌握马尔可夫决策过程 (MDP)
- 理解 Q-learning 和 SARSA 算法
- 了解深度强化学习 (DQN, Policy Gradient)

## 📖 核心概念

### 强化学习 vs 监督学习

| 特性 | 监督学习 | 强化学习 |
|------|---------|---------|
| 反馈类型 | 正确标签 | 奖励信号 |
| 反馈时机 | 即时 | 延迟 |
| 数据来源 | 静态数据集 | 智能体与环境交互 |
| 目标 | 预测正确标签 | 最大化累积奖励 |
| 探索 vs 利用 | 无 | 核心权衡 |

### 核心要素

强化学习包含五个核心要素：

1. **智能体 (Agent)**：做出决策的实体
2. **环境 (Environment)**：智能体交互的外部世界
3. **状态 (State)** $s_t$：环境在时刻 t 的描述
4. **动作 (Action)** $a_t$：智能体在时刻 t 的选择
5. **奖励 (Reward)** $r_t$：环境对动作的即时反馈

### 马尔可夫决策过程 (MDP)

MDP 是强化学习的数学框架，定义为五元组 $\langle S, A, P, R, \gamma \rangle$：

- $S$：状态空间
- $A$：动作空间
- $P(s'|s,a)$：状态转移概率
- $R(s,a)$：奖励函数
- $\gamma \in [0,1]$：折扣因子

**马尔可夫性质**：下一状态只依赖于当前状态和动作，与历史无关。
$$ P(s_{t+1} | s_t, a_t) = P(s_{t+1} | s_t, a_t, s_{t-1}, a_{t-1}, ...) $$

### 价值函数

**状态价值函数** $V^\pi(s)$：从状态 s 开始，遵循策略 π 的期望累积奖励：
$$ V^\pi(s) = \mathbb{E}_\pi\left[\sum_{k=0}^\infty \gamma^k r_{t+k+1} \mid s_t = s\right] $$

**动作价值函数** $Q^\pi(s,a)$：在状态 s 执行动作 a 后的期望累积奖励：
$$ Q^\pi(s,a) = \mathbb{E}_\pi\left[\sum_{k=0}^\infty \gamma^k r_{t+k+1} \mid s_t = s, a_t = a\right] $$

### 贝尔曼方程

贝尔曼最优方程表达了最优价值函数的递归关系：

$$ Q^*(s,a) = R(s,a) + \gamma \sum_{s'} P(s'|s,a) \max_{a'} Q^*(s',a') $$

### Q-Learning 算法

Q-Learning 是一种无模型的时序差分学习算法，使用离策略更新：

$$ Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \left[ r_{t+1} + \gamma \max_{a} Q(s_{t+1}, a) - Q(s_t, a_t) \right] $$

其中 $\alpha$ 是学习率。

### 算法对比

| 算法 | 类型 | 策略 | 适用场景 |
|------|------|------|---------|
| Q-Learning | 基于价值 | 离策略 | 离散动作空间 |
| SARSA | 基于价值 | 在策略 | 离散动作空间 |
| DQN | 深度+价值 | 离策略 | 高维状态空间 |
| Policy Gradient | 基于策略 | 在策略 | 连续动作空间 |
| Actor-Critic | 混合 | 在策略 | 连续+高维 |
| PPO | 混合 | 在策略 | 通用(最常用) |

### 探索与利用权衡

- **探索 (Exploration)**：尝试新动作以获取更多信息
- **利用 (Exploitation)**：选择已知最优动作以最大化奖励

**ε-greedy 策略**：
- 以概率 $1-\varepsilon$ 选择最优动作（利用）
- 以概率 $\varepsilon$ 随机选择动作（探索）

## 💻 代码实现

```python
import numpy as np
from collections import defaultdict

class QLearningAgent:
    """Q-Learning 智能体"""
    
    def __init__(self, n_states, n_actions, learning_rate=0.1,
                 gamma=0.99, epsilon=0.1):
        self.n_actions = n_actions
        self.lr = learning_rate
        self.gamma = gamma
        self.epsilon = epsilon
        self.q_table = defaultdict(lambda: np.zeros(n_actions))
    
    def choose_action(self, state):
        """ε-greedy 动作选择"""
        if np.random.random() < self.epsilon:
            return np.random.randint(self.n_actions)  # 探索
        return np.argmax(self.q_table[state])          # 利用
    
    def learn(self, state, action, reward, next_state, done):
        """Q-Learning 更新"""
        current_q = self.q_table[state][action]
        
        if done:
            target = reward
        else:
            target = reward + self.gamma * np.max(self.q_table[next_state])
        
        # Q-Learning 更新公式
        self.q_table[state][action] += self.lr * (target - current_q)

class GridWorld:
    """简单的网格世界环境"""
    
    def __init__(self, size=5):
        self.size = size
        self.goal = (size-1, size-1)
        self.reset()
    
    def reset(self):
        self.agent_pos = (0, 0)
        return self._get_state()
    
    def _get_state(self):
        return self.agent_pos[0] * self.size + self.agent_pos[1]
    
    def step(self, action):
        # 0:上 1:下 2:左 3:右
        moves = [(-1,0), (1,0), (0,-1), (0,1)]
        r, c = self.agent_pos
        dr, dc = moves[action]
        r = max(0, min(self.size-1, r + dr))
        c = max(0, min(self.size-1, c + dc))
        self.agent_pos = (r, c)
        
        done = self.agent_pos == self.goal
        reward = 10 if done else -1
        return self._get_state(), reward, done

# 训练
env = GridWorld(size=5)
agent = QLearningAgent(
    n_states=25, n_actions=4,
    learning_rate=0.1, gamma=0.99, epsilon=0.1
)

episodes = 500
for ep in range(episodes):
    state = env.reset()
    total_reward = 0
    
    while True:
        action = agent.choose_action(state)
        next_state, reward, done = env.step(action)
        agent.learn(state, action, reward, next_state, done)
        
        state = next_state
        total_reward += reward
        if done:
            break
    
    if (ep + 1) % 100 == 0:
        print(f"Episode {ep+1}, Total Reward: {total_reward}")

print("\n训练完成！")
print("Q-table 示例 (状态0的最优动作):", 
      ['上','下','左','右'][np.argmax(agent.q_table[0])])
```

## 🎯 关键要点
- 强化学习通过与环境交互来学习最优策略
- MDP 是 RL 的数学基础，关键假设是马尔可夫性质
- Q-Learning 是无模型算法，不需要知道环境模型
- ε-greedy 策略平衡探索与利用
- 深度强化学习 = 深度学习 + 强化学习（DQN, PPO）

## ❓ 常见问题

Q: Q-Learning 和 SARSA 的区别是什么？
A: Q-Learning 是离策略的（用 max Q 更新），SARSA 是在策略的（用实际选择的动作更新）。Q-Learning 更激进，SARSA 更保守。

Q: 什么时候用强化学习而不是监督学习？
A: 当没有标注数据，只能通过试错获得反馈时，如游戏、机器人控制、推荐系统等。
